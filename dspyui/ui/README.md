# ui

Gradio UI 组件模块，支持中文界面显示。
包含样式、可复用组件、主应用和各个 Tab 页面。
集成 i18n 国际化翻译系统。

## 文件清单

| 文件 | 地位 | 功能 |
|-----|------|------|
| `__init__.py` | 入口 | 导出 UI 组件 |
| `styles.py` | 辅助 | CSS 样式定义，含语言切换器样式 |
| `components.py` | 辅助 | 可复用的 UI 组件 |
| `language_switcher.py` | 辅助 | 语言切换组件，支持中英文切换 |
| `app.py` | 核心 | 主应用组装，集成语言切换器 |
| `tabs/` | 子模块 | 各 Tab 页面实现，已适配中文 |
