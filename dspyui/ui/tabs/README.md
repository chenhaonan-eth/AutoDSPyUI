# tabs

Gradio Tab 页面模块。
包含编译程序、浏览提示词、测试程序三个主要功能页面。

## 文件清单

| 文件 | 地位 | 功能 |
|-----|------|------|
| `__init__.py` | 入口 | 导出各 Tab 创建函数 |
| `compile_tab.py` | 核心 | Compile Program Tab |
| `browse_tab.py` | 核心 | Browse Prompts Tab |
| `test_tab.py` | 核心 | Test Program Tab |
