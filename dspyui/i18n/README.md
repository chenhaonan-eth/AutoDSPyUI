# 国际化翻译模块

提供统一的文本翻译服务，支持多语言界面。

⚠️ 一旦我所属的文件夹有所变化，请更新我。

## 文件结构

- `__init__.py` - 翻译函数和加载逻辑，核心翻译服务
- `zh_CN.py` - 中文翻译字典，默认语言数据
- `en_US.py` - 英文翻译字典，备用语言数据
- `README.md` - 模块说明文档，架构概述

## 翻译命名空间

- `common.*` - 通用组件翻译
- `compile.*` - 编译程序 Tab 翻译
  - `compile.mlflow.*` - MLflow 集成相关翻译
- `browse.*` - 浏览提示词 Tab 翻译
  - `browse.mlflow.*` - MLflow 模型注册相关翻译
- `test.*` - 测试模型 Tab 翻译


## MLflow 集成翻译

新增 MLflow 相关翻译支持：
- 实验追踪界面文本
- 模型注册功能文本
- 批量推理统计信息
- 错误和成功消息
- UI 按钮和链接文本
